Keep the Beat!

Interactive Learning Website

Description

The goal of this project was to create an interactive learning website. Keep the Beat! is a game aimed at elementary to middle school children that teaches some of the basics of rhythm and sight reading. One plays by clicking the drum on the screen as notes scroll past. The game can be played at jeffklouda.com/learning_game/.

Tools

HTML, CSS, Javascript, Illustrator

The game in action

As you can see, one plays the game by first listening to a simple rhythm and then trying to repeat it by clicking on the drum in the center of the screen. Much of the work for this project went into figuring out the right visual and textual cues to use to tell the player what to do and when to do it.
